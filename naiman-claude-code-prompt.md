# NAIMAN Marketing Splash Page — Claude Code Prompt

Copy and paste the prompt below into Claude Code. Before running it, make sure your image assets are in a folder (e.g., `./assets/images/`) in your project directory.

---

## The Prompt

```
Build a single-page marketing splash site for NAIMAN — a premium cardamom & saffron breath freshener brand targeting Gen Z and millennial Americans. This is a pre-launch page. The brand positions itself as a "digestif pearl" — a post-meal ritual, not a mint. Think: the confidence of a streetwear drop meets the sophistication of an amaro brand.

## Brand Identity

- **Name:** NAIMAN
- **Origin story:** From the Arabic benediction "Na'eeman" (نعيماً) — said after someone showers, gets a haircut, or refreshes. It means "may you enjoy this freshness." Every NAIMAN pearl is a tiny ritual of renewal.
- **Tagline options (pick the one that fits the layout best):**
  - "The after-dinner pearl."
  - "Refresh ritual."
  - "The finishing course."
  - "Be fresh."
- **Product:** Tiny silver-coated cardamom & saffron seeds (peppercorn-sized, pearlescent shimmer). Sugar-free. Botanical. Comes in a premium pocket tin.
- **Target retail:** Trader Joe's, Whole Foods, specialty grocers
- **Target consumer:** 22-38 year olds, food-curious, design-conscious, the kind of person who notices packaging

## Visual Direction

- **Palette:** Coral/burnt orange (#E8573A range) as the dominant color, matte black, warm cream/off-white, metallic silver accents. Look at the image assets for the exact vibe.
- **Aesthetic:** Streetwear editorial meets luxury apothecary. NOT clinical, NOT generic wellness. Think: if Aesop and Supreme had a baby that sold breath fresheners.
- **Typography:** Use a distinctive display/script font for "NAIMAN" (something with personality — brush script, editorial serif, or custom-feeling logotype). Pair with a clean, modern sans-serif for body copy. Load from Google Fonts or CDN. Do NOT use Inter, Roboto, or Arial.
- **Photography:** I have image assets in `./assets/images/` — use whichever ones work best for the layout. These are Midjourney-generated editorial/lifestyle/product shots. You don't need to use all of them. Pick the strongest ones that serve the layout.

## Page Structure (single scroll)

1. **Hero** — Full-bleed, bold. The brand name NAIMAN huge. A compelling hero image or image pair. The tagline. A single CTA: email waitlist signup (just an email input + submit button — store nothing, it's a placeholder).

2. **The Ritual** — Short section explaining the concept: "The pearl you reach for after." Position it as the modern digestif — what you do after coffee, after dinner, after a date. 2-3 short lines of copy max. Support with a lifestyle image.

3. **The Product** — What's in it: cardamom, saffron, sugar-free, silver-coated, pocket tin. Keep it ingredient-forward and sensory. Use a product/detail image. Don't make it a spec sheet — make it feel like you can taste it.

4. **The Origin** — Brief story block: the Na'eeman tradition, the Silk Road spice heritage, the ritual of freshness across cultures. This is the soul of the brand. 3-4 sentences max.

5. **Waitlist CTA** — Repeated email capture. "Be the first to taste it." or similar.

6. **Footer** — Minimal. Brand name, "Chicago, IL", placeholder social icons (Instagram, TikTok).

## Technical Requirements

- **Single HTML file** with inline CSS and JS (no build step, no framework). This needs to be deployable by dragging into Netlify or Vercel.
- **Responsive** — must look incredible on mobile (this audience lives on their phones) AND desktop.
- **Performance** — lazy load images, optimize for fast first paint.
- **Animations** — tasteful scroll-triggered reveals, a smooth hero entrance. Nothing cheesy. CSS-only preferred. The vibe is confident and unhurried, not bouncy.
- **Image handling** — reference images from `./assets/images/` using relative paths. Size them appropriately with CSS (object-fit, etc). Use max 4-6 images total on the page.
- **No stock placeholder images or emoji**. Only use the actual assets I've provided, or leave image slots empty with a styled placeholder div if none fit.
- **The page should feel like it was designed by a human at a boutique branding agency, not generated by AI.** Every spacing choice, every font weight, every color value should feel intentional.

## Copy Tone

Write the marketing copy directly — don't use lorem ipsum. The voice is:
- Confident but not arrogant
- Sensory and evocative (you should almost taste the cardamom reading it)
- Short sentences. No filler. Every word earns its place.
- Lowercase aesthetic is fine if it fits the design
- NO exclamation marks. NO "revolutionizing." NO "game-changing." NO startup buzzwords.

## What I DON'T want

- Generic gradient hero sections
- Rounded-corner card grids
- Purple/blue SaaS aesthetics
- Cookie-cutter Tailwind templates
- Overly busy layouts — this is luxury, give it room to breathe
- Multiple pages or routing
- Any JavaScript framework or build tooling

Look at the images in `./assets/images/` first to understand the visual language, then design the page around the strongest assets.
```

## Tips

- If you have 15+ images, pre-select your top 8-10 before running the prompt so Claude doesn't waste context window on weak assets
- Name your files descriptively (e.g., `hand-holding-pearls-coral-bg.png` not `image_23.png`) so Claude can make smart selections
- After the first generation, iterate: "make the hero bolder", "swap the ritual section image", "tighten the spacing on mobile", etc.
- If you want to add an actual Mailchimp/ConvertKit form later, ask Claude to wire up the email input to the API
